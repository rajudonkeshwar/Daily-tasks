What Is AWS Transit Gateway?
  
  AWS Transit Gateway (TGW) is a central network hub that allows you to connect multiple VPCs, on-premises networks, 
  and VPNs using a hub-and-spoke model.
  
  Instead of connecting networks to each other one by one, everything connects to the Transit Gateway.
